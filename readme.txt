== Description ==

Protect your Wordpress from several security attacks (XSS, SQL Injection, etc) using an Expose's modified version (https://github.com/enygma/expose). 

This plugin ever will be free, never Serviceware or Trailware. We will not create a free lite version with limited options an other one non-free with full options.

== Installation ==

1. Download and install.
2. Activate the plugin.
3. Configure general options: contact email, IDS state (If we must stop the malicious request furthermore logging it), IPS state (blocking state) and minimun treshold.
